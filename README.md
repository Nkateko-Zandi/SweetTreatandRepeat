# 🍬 Sweet Treat & Repeat

Sweet Treat & Repeat is a colourful and interactive sweet treats website designed to showcase delicious candy products in a fun and modern way.

The website includes:
- A responsive landing page
- About page
- Product showcase page
- Interactive contact side panel
- Falling sweets animation
- Clickable product image gallery with lightbox preview

---

# ✨ Features

## 🏠 Home Page
- Hero section with product image
- Gradient aesthetic background
- Navigation bar
- Button linking to product page

## 📖 About Page
- Company introduction
- Animated falling sweets background
- Responsive layout
- Modern rounded container design

## 🍭 Product Page
- Product details section
- Product highlights
- WhatsApp ordering button
- Interactive image gallery
- Lightbox popup with image navigation

## 📱 Contact Panel
- Slide-out contact panel
- TikTok link
- Instagram link
- WhatsApp ordering contact
